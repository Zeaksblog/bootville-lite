    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-bootville-lite" class="anchor" href="#bootville-lite" aria-hidden="true"><span class="octicon octicon-link"></span></a>Bootville Lite</h1>

<p>Bootville lite is a clean responsive WordPress theme created with Underscores starter theme by Automattic and Bootstrap framework and Font Awesome.</p>

<h2>
<a id="user-content-features" class="anchor" href="#features" aria-hidden="true"><span class="octicon octicon-link"></span></a>Features</h2>

<ul class="task-list">
<li>Clean responsive layout, works on any screen size</li>
<li>Two menu areas, one in the header and one in the footer</li>
<li>4 Widget areas, one in the right sidebar, three in the footer</li>
<li>Display featured image above each post</li>
<li>Social network icons by Font Awesome in the header via custom menu</li>
<li>All Boostrap components available via several shortcode plugins or HTML</li>
<li>Header area will display the free CPT Bootstrap Carousel plugin without adding any code, or header image or no image</li>
<li>Easily add Bootswatch color schemes or create child themes</li>
<li>Portfolio page template upgrade available</li>
<li>Full theme version available with many useful options including page templates, 200+ Google Fonts, homepage template and sidebar layouts</li>
<li>GPL 2 licensed</li>
</ul>

<p>Visit <a href="http://zeaks.org">http://zeaks.org</a> for questions on modifying and other Bootville resources.</p>
