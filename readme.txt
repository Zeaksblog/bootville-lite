/* Description of Theme */
/*-----------------------------------------------------------------------------------*/

Bootville-lite is a Light weight Responsive WordPress theme that looks great at any screen size. It is professional, clean and Responsive, built using Underscores starter theme, Bootstrap and Font Awesome. It comes with custom header support, support for one of the most popular Bootstrap Carousel sliders and Bootstrap shortcodes plugin (see theme info page in admin), 4 widget areas, header social icons via custom menus and more.
For questions, comments, bug reports or feature requests visit theme forum at http://www.themes.zeaks.org/forums/
