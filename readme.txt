/* Theme Features */
/*-----------------------------------------------------------------------------------*/

daisy-blue is full responsive wordpress theme which has the following features

1  = Custom Header.
2  = Custom Background.
3  = Custom Fonts.
4  = Bootstrap 3 all features.
5  = SEO Friendly.
6  = Threaded Comments.
7  = Responsive Layout.
8  = Flexible Layout.
9  = Full width template
10 = Right sidebar.
11 = Two Columns.

Theme support only dropdown menu no further sub menus are supported from dropdown menu.

/*-----------------------------------------------------------------------------------*/
/* Description of Theme */
/*-----------------------------------------------------------------------------------*/

daisy-blue is the Most Light weight Responsive WordPress theme. It is professional, clean, & Responsive WordPress Theme, built using Bootstrap, Root, SASS & WordPress Code standards which makes the daisy-blue optimized for all devices like iPhone, Tablet, mac book, desktop & all other wide and small devices it is supported with custom Header & Background to configure logo and custom background.

For questions, comments, bug reports or request features visit theme forum at http://oopthemes.com/forums/forum/themes-forum/
